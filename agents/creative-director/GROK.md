# Creative Director Agent

## Overview

Provides creative direction, brand design, and visual content strategy.

## Capabilities

- **Brand Identity**: Create brand identities
- **Visual Design**: Design visual assets
- **Creative Strategy**: Develop creative strategies
- **Design Systems**: Build design systems
- **Creative Review**: Review and critique creative work

## Usage

```python
from agents.creative-director.agent import CreativeDirectorAgent

agent = CreativeDirectorAgent()
identity = agent.create_brand_identity(brand="new-brand")
```

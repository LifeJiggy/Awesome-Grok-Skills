# Debugging Agent

## Overview

Helps debug code issues, errors, and performance problems.

## Capabilities

- **Error Analysis**: Analyze error messages
- **Code Debugging**: Debug code issues
- **Performance Diagnosis**: Diagnose performance issues
- **Root Cause Analysis**: Find root causes
- **Fix Suggestions**: Suggest fixes

## Usage

```python
from agents.debugger.agent import DebuggingAgent

agent = DebuggingAgent()
analysis = agent.analyze_error(error_message="TypeError: None")
```

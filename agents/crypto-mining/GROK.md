# Crypto Mining Agent

## Overview

Manages cryptocurrency mining operations and optimization.

## Capabilities

- **Mining Operations**: Manage mining rigs
- **Profitability Analysis**: Analyze mining profitability
- **Pool Management**: Manage mining pools
- **Hardware Optimization**: Optimize mining hardware
- **Energy Management**: Optimize energy usage

## Usage

```python
from agents.crypto-mining.agent import CryptoMiningAgent

agent = CryptoMiningAgent()
analysis = agent.analyze_profitability(coin="bitcoin")
```

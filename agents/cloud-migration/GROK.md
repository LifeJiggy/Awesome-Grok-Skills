# Cloud Migration Agent

## Overview

Plans and executes cloud migration strategies and workloads.

## Capabilities

- **Migration Assessment**: Assess workloads for migration
- **Strategy Design**: Design migration strategy
- **Lift and Shift**: Execute lift and shift migrations
- **Modernization**: Modernize applications for cloud
- **Migration Testing**: Test migrated workloads

## Usage

```python
from agents.cloud-migration.agent import CloudMigrationAgent

agent = CloudMigrationAgent()
plan = agent.assess_workloads(servers=["server1", "server2"])
```

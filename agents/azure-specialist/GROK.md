# Azure Specialist Agent

## Overview

Expert in Microsoft Azure cloud services, architecture, and DevOps practices.

## Capabilities

- **Azure Compute**: VMs, App Services, Functions
- **Azure Storage**: Blob, Table, Queue storage
- **Azure Kubernetes**: AKS container orchestration
- **Azure DevOps**: Pipelines, Boards, Repos
- **Azure Security**: Active Directory, security center

## Usage

```python
from agents.azure-specialist.agent import AzureSpecialistAgent

agent = AzureSpecialistAgent()
vm = agent.create_vm(name="azure-vm", size="Standard_B1s")
```
